   ```bash
   designTokens transform config.json
   ```

2. **Replace the CSS variables** in `css/style.css` with your generated tokens:
   ```css
   /* Replace these with your generated CSS custom properties */
   :root {
     /* Your design tokens here */
   }
   ```

## ğŸ”§ Customization

### Changing Colors
Update the CSS custom properties in `:root`:

```css
:root {
  --color-primary: #your-color;
  --color-secondary: #your-color;
  /* ... other colors */
}
```

### Modifying Spacing
Adjust the spacing scale:

```css
:root {
  --spacing-sm: 0.75rem;  /* Increase small spacing */
  --spacing-lg: 2rem;     /* Adjust large spacing */
}
```

### Typography Changes
Update font properties:

```css
:root {
  --font-family-base: 'Your Font', sans-serif;
  --font-size-base: 1.125rem;  /* Larger base font size */
}
```

Access design token utilities via browser console:

```javascript
// Change primary color
DesignTokens.setToken('color-primary', '#ff6b35');

// Apply a theme
DesignTokens.applyTheme('dark');

// Generate random color
DesignTokens.randomColor();
```
