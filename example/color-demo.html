Color Format Conversion in Action

This demo shows how the outputColorFormat: "hex" configuration converts all color tokens to hexadecimal format, regardless of their original format in the token files.

#c8c8c8

Configuration

The color conversion is controlled by the outputColorFormat property in your config file:

{ "outputColorFormat": "hex", "keepReferences": true, "transforms": [ { "name": "color-demo", "from": "example/tokens-colors", "to": [ {"as": "css", "to": "example/color-output/css"}, {"as": "scss", "to": "example/color-output/scss"}, {"as": "mjs", "to": "example/color-output/js"}, {"as": "json", "to": "example/color-output/json"} ] } ] }

Supported formats:

  • "hex" - Convert to hexadecimal (#rrggbb, #rrggbbaa)
  • "rgb" - Convert to RGB/RGBA format
  • "hsl" - Convert to HSL/HSLA format
  • "auto" - Preserve original formats (default)

Generated CSS Output

Here's what the CSS output looks like with color conversion applied:

:root { --primary-hex: #ff6b35; --primary-rgb: rgb(255, 107, 53); --primary-hsl: hsl(14, 100%, 60%); --primary-transparent: #ff6b3580; --secondary-hex: #007bff; --secondary-rgb: rgb(0, 123, 255); --secondary-hsl: hsl(207, 100%, 50%); --semantic-background: var(--color-primary-hex); --semantic-text: var(--color-secondary-hex); --semantic-border: #c8c8c8; }

Token Input

The original tokens were defined in various formats:

{ "color": { "primary": { "hex": { "$value": "#ff6b35" }, "rgb": { "$value": "rgb(255, 107, 53)" }, "hsl": { "$value": "hsl(14, 100%, 60%)" }, "transparent": { "$value": "#ff6b3580" } }, "secondary": { "hex": { "$value": "#007bff" }, "rgb": { "$value": "rgb(0, 123, 255)" }, "hsl": { "$value": "hsl(207, 100%, 50%)" } }, "semantic": { "background": { "$value": "{color.primary.hex}" }, "text": { "$value": "{color.secondary.hex}" }, "border": { "$value": "rgb(200, 200, 200)" } } } }
