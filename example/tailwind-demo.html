Automatically generate Tailwind v3 JavaScript configs and v4 CSS @theme directives from your design tokens

Supported Output Formats

Tailwind v3 JavaScript Configuration

Use tailwind-config as the output format to generate traditional Tailwind configuration files.

{ "transforms": [ { "as": "tailwind-config", "to": "config" } ] }

Tailwind v4 CSS @theme Directive

Use tailwind-theme as the output format to generate modern CSS with @theme directives.

{ "transforms": [ { "as": "tailwind-theme", "to": "css" } ] }

Token Mapping

Design tokens are automatically categorized and mapped to appropriate Tailwind theme sections:

Token Pattern Tailwind v3 Section Tailwind v4 CSS Variable
color-*, bg-*, text-* theme.colors --color-*
spacing-*, space-* theme.spacing --spacing-*
font-size-* theme.fontSize --font-size-*
font-family-* theme.fontFamily --font-family-*
border-radius-* theme.borderRadius --border-radius-*

Integration with Color Format Conversion

Tailwind outputs respect the outputColorFormat configuration, ensuring consistent color formats across your theme:

{ "outputColorFormat": "hex", "transforms": [ { "as": "tailwind-config", "to": "config" }, { "as": "tailwind-theme", "to": "css" } ] }
